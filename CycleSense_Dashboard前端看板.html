使用指南
配置 AI 并开始使用
配置步骤
配置一次，以后正常聊天就可以。
1
启动 CycleSense
打开本地服务，让看板能读到今天的数据。
cyclesense start
2
连接采集设备
设备通电后自动上报。
3
把 CycleSense 加到 Claude
在 Claude Desktop 配置里加入这段：
"cyclesense": { "command": "cyclesense", "args": ["mcp"] }
4
重启 Claude，说"今天状态"
AI 会显示你的状态卡。
试试对 AI 说这些
越像平常说话越好，不需要懂术语。
  • "今天状态"先看状态卡
  • "帮我把今天安排轻一点"按余量排任务
  • "精力卡片"查看今日精力签
  • "今天适合谈那件重要的事吗？"看沟通时机
  • "今天来月经了"更新周期记录
  • "上次什么方法有效？"查历史策略

你可能想知道
把容易焦虑的问题说清楚。

年龄和经期必须填吗？
不是。填了会更贴近你的身体节奏，不想填也可以只看今日余量。

精力低会不会被评价？
不会。AI只会帮你调整安排，不会说你不够好。

数据安全吗？
所有数据在你电脑本地，不上传任何云端。

后端需要一直开着吗？
是的。关闭后AI无法获取你的状态。下次用 cyclesense start 启动。
